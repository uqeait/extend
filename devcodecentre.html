Stop and Read! How to use this table generator when pasting a table

If you have created a table in MS Word or Google Docs:

  1. Select the entire contents of the table
  2. Paste them into an Excel Spreadsheet/Google Sheet
  3. Copy the entire contents of the table from Excel/Sheets and paste them here

Note: If you try to paste here directly from MS Word your entire table will likely be deleted. The code here attempts to strip out all the extraneous attributes and empty elements, and inevitably in the process the entire contents of the table will be deleted.

You can also try pasting your MS Word table to HTML Cleaner and using this tool to clean up your table's code before pasting it here

Stop and Read! How and when to use accordions

Accordions should be used:

  • for FAQs about a topic or workshop
  • when the context calls for users to focus in on separate information that is all linked - e.g. a summary of a case study that has Histology, Pathology etc. under accordion headings.

Accordions should not be used:

  • to save scrolling a long page. If the page is set out well with headings and subheadings, users will scroll. If it's long chunks of text with no subheadings to guide the user, then start to reformat the text to include these rather than turn to accordions.
  • for definitions of terminology behind a click. This is irritating behaviour to have to click a button to get the information all the time.

Nielson Norman Group, global leaders in user experience offer this article on accordions for those interested in further reading.
